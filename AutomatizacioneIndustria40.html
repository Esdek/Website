Arquitectura de control

A continuación se muestra la arquitectura que maneja el proyecto. Se incluyen los elementos de nivel de proceso (Siemens NX y ABB RobotStudio), nivel de control (controlador del laboratorio de la UNAL, Studio5000 y RSLinx para realizar la conexión entre estos elementos), nivel de supervisión (interfaz diseñada con Ignition designer) y una nube donde se ejecuta el servidor de Ignition para mantener la información en un punto centralizado.
